# Marketing Copywriting Avançado ✍️

Skill treinada com as melhores técnicas de persuasão em texto (Copywriting) para gerar conversão (vendas, leads, cliques).

## Frameworks Nativos

### 1. Sistema AIDA (Atenção, Interesse, Desejo, Ação)
Perfeito para páginas de vendas e e-mails longos.
- **Atenção**: Gancho (Hook) que trava a rolagem da tela.
- **Interesse**: Introdução do problema que o cliente já sabe que tem.
- **Desejo**: Apresentação da solução (seu produto) transformando características em **benefícios emocionais**.
- **Ação**: Call To Action (CTA) claro e único. Evite fricção.

### 2. Metodologia PAS (Problem, Agitation, Solution)
Perfeito para Anúncios (Facebook Ads / Instagram Ads):
- **Problema**: Identifique a dor. ("Gastando horas para editar um projeto Xcode?")
- **Agitação**: Pressione a ferida. ("Cada minuto copiando e colando código é tempo perdido não faturando.")
- **Solução**: O alívio. ("A automação X recorta seu trabalho pela metade...")

### 3. Técnicas de Título (Hooks)
O título é 80% do sucesso.
- O segredo de [X] para conseguir [Y] sem [Z].
- Exemplo: "O segredo dos Devs Seniores para publicar apps iOS sem abrir o Xcode."

## Restrições de Tom de Voz
- Nunca use palavras de enchimento ("muito", "extremamente").
- Fale diretamente com o leitor usando "Você".
- Evite linguagem passiva. Verbos fortes sempre.
