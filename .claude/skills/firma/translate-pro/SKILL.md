# Translate-Pro (Localization)

Master accurate translation that preserves meaning, formatting, and cultural context.

## Core Rules
- **Formatting Preservation**: Never translate content inside code blocks, HTML, or markdown syntax. Preserve placeholders like `{name}` or `{{variable}}`.
- **Content Rules**: Don't translate proper nouns, brand names, technical terms, or URLs.
- **Language-Specific**: Use correct plural forms and gender agreement (Spanish/French/German).
- **Cultural Adaptation**: Adapt idioms and expressions. Convert units (miles↔km). Adjust date formats.

## Quality Control
- Read full context before translating.
- Ensure consistent voice across the entire document.
- Verify all formatting elements remain functional.
