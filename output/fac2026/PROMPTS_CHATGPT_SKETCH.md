Details: Show fabric folds gently. Render the figure's face with peace and resolution.

Mood: Profound stillness. Peace after storm. Resolution and acceptance. The absence

## 📋 INSTRÇÕES PARA USAR NO CHATGPT:

1. **Abra ChatGPT:** https://chatgpt.com
2. **Modelo:** Certifique-se que está usando um modelo com geração de imagem (DALL-E 3)
3. **Para cada imagem:** Cole o prompt completo abaixo
4. **IMPORTANTE:** Adicione ao final de CADA prompt a instrução:
   **"CRITICAL: NO TEXT, LETTERS, WORDS, ANNOTATIONS, OR LABELS VISIBLE IN THE IMAGE. This is a pure visual/sketch only."**
5. **Variações:** Peça 2-3 variações de cada imagem
6. **Seleção:** Escolha a que melhor corresponde à visão do projeto
7. **Salvamento:** Baixe/salve as imagens
