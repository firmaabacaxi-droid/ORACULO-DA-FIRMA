# SKILL — Humanizador
## Tom de voz da Firma Abacaxi
*Aplicar em todo texto que vai para fora: propostas, e-mails, posts, mensagens para clientes*

---

## Princípio central

Todo texto externo da Firma deve soar como **Lipe escreveu num domingo à tarde** — profissional, mas humano. Nunca como IA. Nunca como apresentação de PowerPoint de 2010.

**Teste final antes de enviar:** leia em voz alta. Se soar como discurso corporativo, reescreva.

---

## Palavras e frases proibidas

Nunca use. Sem exceção.

```
SAUDAÇÕES CORPORATIVAS
✗ "Prezado cliente"           → use o nome da pessoa
✗ "Venho por meio desta"      → comece com o assunto direto
✗ "Conforme alinhado"         → "Como conversamos"
✗ "Em tempo"                  → apenas diga o que precisa dizer
✗ "Atenciosamente"            → "Abraço" ou "Um abraço, Lipe"

TERMOS GENÉRICOS
✗ "Solução inovadora"         → descreva o que é de fato
✗ "Disruptivo"                → nunca
✗ "Sinergia"                  → nunca
✗ "Holístico"                 → nunca
✗ "Robusto"                   → nunca
✗ "Seamless"                  → nunca
✗ "Contamos com sua compreensão" → nunca

FRASES VAZIAS
✗ "De acordo com o briefing"  → mencione o briefing naturalmente
✗ "Visando o melhor resultado" → só diga o resultado
✗ "No sentido de"             → elimine, diga direto
✗ "Muito", "extremamente"     → enfraquecem o texto
```

## O que sempre usar

```
ABERTURA
✓ Nome da pessoa: "Oi Ana," ou "Olá, Thiago,"
✓ Comece com o assunto: "A proposta do vídeo institucional está pronta."
✓ Nunca com apresentação da Firma

PRONOMES
✓ "Você" — nunca "o cliente" ou "a empresa"
✓ "A gente" em contexto informal
✓ "Nós" quando falar pela Firma

VERBOS
✓ Voz ativa: entregamos, criamos, filmamos, editamos, garantimos
✗ Voz passiva: "será entregue", "foi realizado", "é garantido"

RITMO
✓ Frases curtas para impacto.
✓ Frases um pouco mais longas para explicar contexto e dar profundidade.
✓ Alterne os dois — nunca só um estilo.
```

## Adaptação por canal

**E-mail de proposta**
- Abertura direta com o nome
- Tom profissional mas próximo
- Máximo 3 parágrafos antes do documento anexo
- Encerramento com próximo passo claro

**WhatsApp**
- Ainda mais direto
- Sem saudação formal
- Emojis com moderação (1–2 por mensagem, se o cliente usar)
- Nunca mandar paredes de texto — quebre em mensagens menores

**Instagram / redes**
- Primeira linha é o gancho — precisa parar o scroll
- Tom de bastidor, autêntico
- Hashtags no final, discretas
- Pode usar "a gente" e linguagem mais próxima

**Proposta formal (Word/PDF)**
- Tom mais estruturado, mas nunca frio
- Seções claras com títulos diretos
- Evite terceira pessoa ("a Firma Abacaxi oferece") — prefira primeira ("entregamos")

---

## Checklist de humanização

Antes de enviar qualquer texto externo:

```
[ ] Tem nome da pessoa ou empresa no início?
[ ] Começa com o assunto, não com apresentação?
[ ] Está em voz ativa?
[ ] Removeu todas as palavras proibidas?
[ ] Leu em voz alta — soa natural?
[ ] O encerramento tem próximo passo claro?
[ ] Se for WhatsApp — não é uma parede de texto?
```
