## PILARES DE CONTEÚDO DA FIRMA

Todo conteúdo da Firma Abacaxi se encaixa em um destes pilares:

```
1. BASTIDORES — making-of, processo criativo, vida de set
   "Mostra que somos humanos e que o trabalho tem profundidade"

2. RESULTADO — projetos concluídos, cases, antes/depois
   "Prova do trabalho bem feito"

3. EDUCATIVO — dicas de produção, câmera, iluminação, negócio
   "Posiciona como referência técnica e criativa"

4. VALORES — natureza, arte, diversidade, sustentabilidade
   "Mostra quem somos além do trabalho"

5. NEGÓCIO — bastidores da Firma, crescimento, decisões
   "Transparência e autoridade como empreendedores"
```

---

## REPROPÓSITO DE PROJETO

Quando terminar um projeto, extraia:

```
De 1 projeto audiovisual, a Firma pode gerar:

→ 1 post de resultado (o vídeo final ou trecho)
→ 1–3 posts de bastidores (fotos/vídeos do set)
→ 1 post educativo (técnica usada no projeto)
→ 1 story de processo (antes/durante/depois)
→ 1 post de valores (se o projeto envolver causa social)
→ LinkedIn: case completo em texto
```

## CRIAÇÃO DE POST — estrutura por plataforma

### Instagram Feed

```
LINHA 1 (gancho) — para o scroll em 1 segundo
Fórmula: [resultado surpreendente] ou [pergunta provocativa] ou [afirmação contraintuitiva]
Exemplos:
✓ "3 dias. 12 locações. 1 documentário que vai te surpreender."
✓ "A maioria das produções erra nesse detalhe de iluminação."
✓ "Filmamos numa comunidade indígena no Cerrado e foi transformador."

CORPO (2–4 linhas)
Contexto rápido do que está sendo mostrado.
Voz: próxima, como se contasse pra um amigo.

CTA (1 linha)
Uma ação. Nunca duas.
Exemplos: "Comenta o que achou 👇" / "Salva pra lembrar" / "Segue pra ver o resultado"

HASHTAGS (no final, discretas)
5–10 hashtags relevantes. Nunca na primeira linha.
```

### Instagram Stories

```
Frame 1: Gancho (pergunta ou afirmação)
Frame 2–4: Desenvolvimento (texto curto + imagem/vídeo)
Frame 5: CTA ou enquete
Duração: máximo 5 frames por sequência
```

### LinkedIn

```
Linha 1: Gancho profissional
Parágrafos curtos (1–3 linhas cada)
Quebras de linha frequentes — LinkedIn pune paredes de texto
Tom: mais formal que Instagram, mas ainda autêntico
Foco: resultado, aprendizado, business
CTA: "O que você acha?" / "Já passou por isso?"
Hashtags: 3–5, no final
```

### YouTube (descrição)

```
Primeiras 2 linhas: resumo do vídeo (aparecem antes do "ver mais")
Corpo: contexto completo do projeto
Links: site, redes, contato
Timestamps se o vídeo for longo
Tags: relevantes para o nicho
```

## CALENDÁRIO EDITORIAL — estrutura mensal

```
Frequência mínima recomendada:
→ Instagram Feed: 3–4x por semana
→ Stories: diário (quando possível)
→ LinkedIn: 1–2x por semana
→ YouTube: 1x por mês (projeto completo ou making-of)

Distribuição por pilar (semana tipo):
Segunda: Resultado (projeto concluído)
Quarta: Educativo (dica técnica)
Sexta: Bastidores (making-of / processo)
Stories: diário, leve, espontâneo
```

Formato do calendário:

```markdown
## Calendário Editorial — [Mês/Ano]

| Data | Plataforma | Pilar | Conteúdo | Status |
|---|---|---|---|---|
| [dd/mm] | Instagram | Resultado | Post: [projeto X concluído] | A criar |
| [dd/mm] | LinkedIn | Negócio | Case: [projeto Y] | A criar |
| [dd/mm] | Instagram | Educativo | Dica: [técnica Z] | Rascunho |
```

## TRANSCRIÇÃO DE ÁUDIO/VÍDEO

Se precisar transcrever material (making-of, entrevistas, reuniões):
→ Use openai-whisper (disponível no sistema após instalação)
→ Comando: `whisper [arquivo] --language Portuguese`
→ Use a transcrição como base para posts e legendas
