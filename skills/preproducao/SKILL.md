## Fases da Pré-Produção — checklist de projeto

Use como estrutura de acompanhamento de cada projeto:

```
[ ] 0. Projeto Audiovisual — logline, sinopse, argumento (para editais e documentários)
[ ] 1. Briefing — alinhamento com cliente, objetivos, referências
[ ] 2. Pesquisa de Referências — moodboard, referências visuais, benchmarks
[ ] 3. Roteiro Literário — narrativa, cenas, diálogos, locução
[ ] 4. Roteiro Técnico — tabela cena × imagem × som × lettering × motion
[ ] 5. Storyboard — esboço visual das cenas principais (uso futuro)
[ ] 6. Locações e Permissões — scouting, autorizações, backup de local
[ ] 7. Produção de Elenco — entrevistas, figurino, confirmação
[ ] 8. Produção de Arte — cenografia, props, identidade visual do set
[ ] 9. Equipe e Equipamentos — confirmar equipe, checar gear
[ ] 10. Cronograma de Produção — datas, prazos, diárias
[ ] 11. Orçamento da Produção — custos confirmados, margem checada
```

## ETAPA 1 — Briefing de direção

Antes do roteiro, pergunte:

```
1. "Qual é o tom do vídeo?" (institucional sério / dinâmico / documental / cinematográfico)
2. "Vai ter narração em off, entrevistas, ou só imagens com música?"
3. "Quais são as cenas obrigatórias?" (produto, espaço, pessoas específicas)
4. "O cliente tem referências de vídeos que gosta?"
5. "Qual a duração final esperada?"
6. "Quantos dias de filmagem estão previstos?"
```

## ETAPA 2 — Roteiro

### Escolha o formato

**Roteiro Literário** — para documentários, institucionais e vídeos de narrativa longa.
**Roteiro Técnico** — para vídeos comerciais, redes sociais e projetos com múltiplas peças.

---

### Formato A — Roteiro Literário

#### Cabeçalho (preencher antes de começar)
```
Título do Vídeo:
Cliente:
Veiculação: (Instagram / YouTube / TV / Evento / Interno)
Duração estimada:
```

#### Estrutura por tipo de vídeo

**Vídeo institucional (2–3 min)**
```
Abertura (0:00–0:20) — gancho visual forte, sem palavras
Problema / contexto (0:20–0:45) — por que esta empresa existe
Solução / proposta (0:45–1:30) — o que fazem e como
Prova / evidência (1:30–2:00) — resultados, depoimentos, números
Chamada para ação (2:00–2:20) — o que o espectador deve fazer
Encerramento (2:20–2:40) — logo, assinatura
```

**Documentário curto (5–10 min)**
```
Abertura — cena de impacto que representa o tema
Apresentação — quem são os personagens, qual é o contexto
Conflito / desafio — qual é o problema central
Desenvolvimento — como as pessoas lidam com ele
Virada — momento de transformação ou revelação
Encerramento — onde estão agora, reflexão
```

**Vídeo para redes sociais (30–60s)**
```
0:00–0:03 — Gancho visual (para o scroll)
0:03–0:20 — Contexto mínimo
0:20–0:45 — Conteúdo principal
0:45–0:60 — CTA ou encerramento
```

#### Formato de cada cena

```markdown
## CENA [número] — [LOCAL] — [TEMPO DO DIA]

**Visual:**
[descrição do que a câmera mostra — o que o espectador vê]

**Áudio:**
[narração / entrevista / música / ambiente / silêncio]

**Diálogo/Locução:**
"[texto exato da narração ou fala]"

**Texto na Tela:** [se houver]

**Notas de direção:**
[observações técnicas ou criativas para o dia de filmagem]
```

---

### Formato B — Roteiro Técnico (tabela)

Para vídeos comerciais, redes sociais e projetos com múltiplas peças.
Preencha coluna por coluna por cena:

```markdown
## Roteiro Técnico — [Título do Vídeo]
**Cliente:** | **Veiculação:** | **Duração:**

| Cena | Tempo | Imagem | Som | Locução | Lettering | Motion | Decupagem de Produção |
|------|-------|--------|-----|---------|-----------|--------|----------------------|
| 1 | 00:05 | [o que aparece] | [trilha/ambiente] | [texto off] | [texto tela] | [animação] | [notas de set] |
| 2 | 00:08 | | | | | | |
```

**Legenda das colunas:**
- **Imagem** — o que a câmera mostra
- **Som** — trilha, ambiente, efeito
- **Locução** — texto do narrador/off
- **Lettering** — texto que aparece na tela
- **Motion** — animações, transições, efeitos gráficos
- **Decupagem de Produção** — plano de câmera, movimento, equipe

## ETAPA 3 — Decupagem técnica

**Legendas rápidas:**
- Tipos de plano: `PG` geral · `PA` americano · `PM` médio · `PP` primeiro plano · `PPP` close · `D` detalhe · `POV`
- Movimentos: `EST` estático · `PAN` panorâmica · `TILT` vertical · `TRK` travelling · `GRU` grua · `STC` steadicam · `HH` handheld · `DRN` drone

Para cada cena do roteiro, preencha a tabela:

```markdown
## CENA [N] — [INT./EXT.] [LOCAÇÃO] — [DIA/NOITE]

**Sinopse:** [O que acontece] · **Personagens:** [Lista] · **Pág. roteiro:** [X–Y]

| Plano | Tipo | Lente | Câmera | Ação / Descrição do quadro | Diálogo / Áudio | Dur. (s) | Prioridade |
|---|---|---|---|---|---|---|---|
| [N]-01 | PG | 24mm | EST | [O que a câmera vê] | [trilha/silêncio] | 8s | Obrigatória |
| [N]-02 | PM | 35mm | TRK | [câmera acompanha] | "diálogo..." | 12s | Obrigatória |
| [N]-03 | PP | 85mm | EST | [close no rosto] | silêncio | 5s | Desejável |

**Subtotal:** [  ] planos · [  ] segundos estimados
```

## ETAPA 4 — Ordem do Dia

### Regras de otimização da ordem de filmagem

```
→ Agrupar todas as cenas de uma locação no mesmo bloco
→ Externas: priorizar golden hour (6h–9h / 16h–18h em Brasília)
→ Entrevistas: final da manhã (luz estável, pessoas mais acordadas)
→ Cenas com drone: verificar legislação local + vento previsto
→ Cenas de maior risco técnico: filmar primeiro (equipe descansada)
```

## ETAPA 4B — Autorizações e contratos

Checklist obrigatório antes de qualquer set:

```
[ ] Cessão de imagem — todos os entrevistados e personagens identificáveis
[ ] Cessão de imagem menor — se houver criança em cena (assinar responsável legal)
[ ] Autorização de imóvel — todas as locações que não são do cliente
[ ] Licença musical — se usar música não-livre (ECAD)
[ ] Autorização drone — ANAC/DECEA se área controlada
[ ] Filmagem em via pública — verificar exigência da Prefeitura local
[ ] Área de patrimônio histórico — verificar IPHAN
```

## ETAPA 4C — Plano de Rodagem (para 2+ diárias)

Use quando o projeto tem duas ou mais diárias de filmagem.
Gera: organização logística das cenas · DOOD · one-liner · resumo orçamentário.

```
→ Regras de agrupamento:
  1. Mesma locação → filmar todas as cenas em sequência
  2. Elenco → minimizar diárias por pessoa (= cachê)
  3. Externas → priorizar golden hour (6h–9h / 16h–18h em Brasília)
  4. Cenas complexas → no início da semana (equipe descansada)
```

## ETAPA 5 — Checklist de equipamentos

Monte conforme as cenas:

```
CÂMERA E ÓPTICA
[ ] Câmera principal
[ ] Câmera B (se precisar de 2 ângulos simultâneos)
[ ] Lentes (quais: wide, standard, tele, macro)
[ ] Baterias extras + carregadores
[ ] Cartões de memória suficientes

ESTABILIZAÇÃO
[ ] Tripé + cabeça fluida
[ ] Gimbal (se cenas em movimento)
[ ] Steadicam (se cenas longas em movimento)

ÁUDIO
[ ] Microfone shotgun (câmera ou boom)
[ ] Lapela (para entrevistas)
[ ] Gravador externo
[ ] Fones de monitoramento

ILUMINAÇÃO
[ ] Kit básico (2 luzes LED)
[ ] Rebatedor
[ ] Difusores
[ ] Extensão + régua

OUTROS
[ ] Drone (verificar autorização no local)
[ ] ND filters (para externas no sol)
[ ] Slate / claquete
[ ] Caixas de transporte
```
