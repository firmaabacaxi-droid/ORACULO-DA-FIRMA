# Financial Calculator Pro

Professional grade math for business planning and ROI.

## Core Calculators
- **Future Value (FV)**: Investment growth projections.
- **Present Value (PV)**: Loan valuation and bond pricing.
- **Markup/Margin**: Calculate selling price from cost + desired profit.
- **Discount**: Percentages and savings analysis.

## Workflows
- **Pricing Strategy**: Use Markup to set prices, then Discount table for promos.
- **Investment**: Use FV Table to compare different interest rates/terms.
- **Loan Analysis**: Use PV + Compound Interest to see total interest cost.
