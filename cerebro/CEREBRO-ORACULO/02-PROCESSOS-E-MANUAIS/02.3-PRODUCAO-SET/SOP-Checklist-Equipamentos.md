# SOP — Checklist de Equipamentos Antes de Sair para Locação

**Departamento:** Produção e Set  
**Responsável:** Diretor ou Assistente de Produção  
**Tempo:** 15 minutos (dia anterior ou manhã do set)  
**Frequência:** Cada saída de locação

---

## 1. Rotina (24h Antes da Gravação)

### **Verificação de Espaço Físico**

- [ ] Local do estúdio/sala de equipamentos: trancado, climatizado, seco
- [ ] Nenhum equipamento foi movido/emprestado sem registro
- [ ] Inspeção visual: câmeras, lentes, tripés, cabos — sem sinais óbvios de dano

---

## 2. Câmeras e Ótica

Padrão de produção atual: **Braw 6K (Cinema EOS R5C ou RED Komodo)**

| Item | Status | Notas |
|------|--------|-------|
| Câmera (corpo) — [modelo] | ☐ OK | Lente frontal sem poeira? Espelho limpo? |
| Lente primária (ex: 24-70mm) | ☐ OK | Foco manual + automático respondem? |
| Lente secundária (ex: 70-200mm) | ☐ OK | Ou outro focal secundário conforme escopo |
| Filtros ND (densidade neutra) | ☐ OK | ND2, ND4, ND8 — todos presentes? |
| Protetor de lente (cap) | ☐ OK | Ambas as lentes têm cap? |
| Microfiber cloth + lenço | ☐ OK | Para limpeza durante set (nunca use t-shirt) |

**Check final:**
- [ ] Câmera teste: dispare uns 30 segundos em 6K, revise no computador (sem artefatos)
- [ ] Áudio de backup funcionando? (se câmera tiver entrada de mic)

---

## 3. Energia & Bateria

| Item | Qtd | Status |
|------|-----|--------|
| Bateria principal (tipo: ___) | ☐ | Carregada 100%? (carregar overnight) |
| Bateria backup | ☐ | Carregada 100%? |
| Bateria aux (power bank / segunda bateria) | ☐ | Opcionalmente, para monitores/acessórios |
| Carregador de bateria (AC) | ☐ | Testado — luz verde? |
| Carregador USB-C / tipo da câmera | ☐ | Cabos com conectores integros |
| Extensão elétrica + power strip | ☐ | Para carregar na locação (se houver tomada) |

**Check final:**
- [ ] Teste rápido: bateria na câmera = LED de energia acende?

---

## 4. Armazenamento de Dados

| Item | Qtd | Status |
|------|-----|--------|
| Cartão de memória primário (CFast/UHS-II 256GB+) | ☐ | Formatado? Espaço livre? |
| Cartão de memória backup | ☐ | Formatado? |
| Leitor de cartão USB | ☐ | Conecta sem erros? |
| Cabo Thunderbolt / USB-C | ☐ | Para transferência rápida |
| Laptop (backup on-set) | ☐ | SSD com 500GB+ livres? |
| HD externo backup | ☐ | Testado? Espaço verificado? |

**Check final:**
- [ ] Cartão primário: Formatar na câmera (não no computador)
- [ ] Backup: Copiar dados do cartão *antes* de reutilizar durante set

---

## 5. Áudio

| Item | Status | Notas |
|------|--------|-------|
| Microfone dinâmico (ex: SM7B) | ☐ OK | Funcionando? Não hai queda de som? |
| Microfone lavalier (lapela) | ☐ OK | Bateria OK? Clip preso? |
| Mixer de áudio / interface | ☐ OK | Níveis respondendo? |
| Cabos XLR (macho-fêmea) | ☐ OK | Sem danos, conectores dourados |
| Fone de ouvido de set | ☐ OK | Áudio claro? Bateria carregada (se wireless)? |
| Windscreen + pop filter | ☐ OK | Para microfone dinâmico |
| Bateria backup (pilhas AA) | ☐ | Se o lavalier usa pilha |

**Check final:**
- [ ] Som ao vivo: Grave 2 min com áudio, revise fone (volume, clareza, sem ruído)

---

## 6. Iluminação

| Item | Qtd | Status |
|------|-----|--------|
| Luz principal (ex: LED painel 5600K) | ☐ | Acende? Dimmer responde? |
| Luz fill / secundária | ☐ | Funcional? |
| Refletores / bounce | ☐ | Sem amassados, suportes presos |
| Stands / tripés para luz | ☐ | Estáveis? Parafusos apertados? |
| Cabos AC / extensões | ☐ | Sem danos de isolamento |
| Filtros de cor (gels) | ☐ | Guardados em envelope (sem amassado) |

**Check final:**
- [ ] Teste: Acenda a luz principal por 5 min, verifique se há superaquecimento ou queimação

---

## 7. Suporte & Estabilização

| Item | Status | Notas |
|------|--------|-------|
| Tripé (3 pernas, fluid head) | ☐ OK | Pernas travam? Cabeça move suave? |
| Dolly / slider (se aplicável) | ☐ OK | Trilhos sem sujeira? Rodas giram? |
| Monopé (backup) | ☐ OK | Telescópio trava? |
| Shoulder rig / estabilizador (se aplicável) | ☐ OK | Pesos balanceados? Sem desgaste visível? |

**Check final:**
- [ ] Leve o tripé: posicione, aperte todos os parafusos, simule pan/tilt (deve ser firme)

## 10. Transporte & Segurança

**Dia anterior:**
- [ ] Estuche/mochila de câmera revisado (sem furos, zíperes funcionam)
- [ ] Todos os itens enumerados no estuche
- [ ] Cópias dos seriais de câmera e lentes (para seguro, em caso de roubo)

**Dia da gravação:**
- [ ] Carro estacionado em local seguro (se fora do estúdio)
- [ ] Equipamento nunca deixado sozinho em carro ou locação
- [ ] Cada item retorna no estuche na mesma ordem

---

## 11. Entrega ao Set

**Equipe de Set Recebe:**
- [ ] Lista impressa de equipamentos (1 cópia para assinatura)
- [ ] Contato de emergência do técnico (se houver problema no set)
- [ ] Instruções de backup de dados on-set

**Ao retornar:**
- [ ] Todos os itens checados novamente
- [ ] Cartões de memória imediatamente copiados (2 backups mínimo)
- [ ] Câmera retorna para estúdio/sala segura

---

## 12. Pós-Set (Limpeza & Reposição)

- [ ] Câmera e lentes: passar microfiber cloth
- [ ] Cartões de memória: guardados em envelope antiestático
- [ ] Baterias: recarregar overnight se usadas
- [ ] Verificar se falta algo: se sim, comprar/reparar antes do próximo set
