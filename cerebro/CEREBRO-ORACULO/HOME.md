## O que é este vault

Repositório de conhecimento vivo da Firma. Tudo que aprendemos, desenvolvemos e refinamos fica aqui — para que o Oráculo seja cada vez mais preciso e para que Lipe e Jaya não reinventem a roda a cada projeto.

**Fluxo principal:**
```
Projeto finalizado → Oráculo extrai aprendizados → salva aqui → git sincroniza → NotebookLM aprende → Oráculo fica mais inteligente
```

## Integrações ativas

| Sistema | Papel |
|---|---|
| **Oráculo (Claude Code / Antigravity)** | Lê e escreve no vault e atualiza o repositório Git/GitHub automaticamente |
| **NotebookLM** | Indexa o vault via GitHub · gera áudios, resumos, analíticos |
| **Notion** | Operação: projetos, tarefas, financeiro, CRM (pontes de link ativo) |
| **Obsidian Git** | Backup e versionamento automático no GitHub a cada 30 min |
