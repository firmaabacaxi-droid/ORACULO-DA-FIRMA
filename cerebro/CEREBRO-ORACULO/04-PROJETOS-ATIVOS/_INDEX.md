## Diretrizes de Uso

1. Cada projeto tem uma subpasta no formato **`FIRMA-NN-Nome-Em-Kebab`** (sem `#`, número com dois dígitos) — ver convenção em [[ARQUITETURA-SISTEMA]] §6.
2. Os arquivos brutos do projeto (edital, planilhas, anexos, briefing, roteiro) vivem nessa pasta — ela é a **fonte única de verdade** do projeto. Cada doc mantém link para a página do Notion no frontmatter.
3. Quando o projeto for fechado financeiramente e entregue, mover a pasta inteira para `05-ARQUIVO-HISTORICO/` para manter a lista ativa limpa.
