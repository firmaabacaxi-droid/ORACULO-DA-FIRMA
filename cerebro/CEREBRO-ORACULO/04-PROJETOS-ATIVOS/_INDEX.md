## Diretrizes de Uso

1. Cada projeto deve ter uma subpasta no formato `#[Número] Nome-do-Projeto` (ou o nome do edital) para manter o histórico físico.
2. Os arquivos de briefing, roteiro e feedbacks devem viver nessa pasta, mantendo um link direto com a página correspondente do Notion no frontmatter.
3. Quando o projeto for fechado financeiramente e entregue, a pasta correspondente deve ser movida inteiramente para `05-ARQUIVO-HISTORICO/` para manter a lista ativa limpa.
