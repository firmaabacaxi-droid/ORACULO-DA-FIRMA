# Template — Roteiro Técnico
*Formato tabular · para vídeos comerciais, redes sociais e projetos com múltiplas peças*

---

## Quando usar este formato

- Vídeos para redes sociais (Reels, Shorts, Stories)
- Vídeos publicitários / comerciais
- Projetos com múltiplas peças (ex: 3 vídeos diferentes para o mesmo cliente)
- Quando a equipe de pós-produção precisa de referências claras de lettering e motion
- Quando há locução em off definida e precisa ser sincronizada com a imagem

→ Para documentários e vídeos de narrativa: use [[Template-Roteiro-Literario]]

---

## Legendas de referência rápida

**Tipos de plano:** `PG` Plano geral · `PA` Plano americano · `PM` Plano médio · `PC` Plano conjunto · `PP` Primeiro plano · `PPP` Close · `D` Detalhe/Insert · `POV` Ponto de vista

**Movimentos de câmera:** `EST` Estático · `PAN` Panorâmica horizontal · `TILT` Vertical · `TRK` Travelling (trilho) · `GRU` Grua/Crane · `STC` Steadicam · `HH` Handheld · `DRN` Drone · `ZOM` Zoom

## Legenda das colunas

| Coluna | O que registrar |
|--------|----------------|
| **Cena** | Número ou letra da cena/plano |
| **Tempo** | Duração do trecho (MM:SS) |
| **Imagem** | O que a câmera mostra — cena, produto, pessoa, B-roll |
| **Som** | Trilha sonora, efeito de som, som ambiente, silêncio |
| **Locução** | Texto exato do narrador em off |
| **Lettering** | Texto que aparece sobreposto na tela |
| **Motion** | Animações, transições, efeitos gráficos, lower thirds |
| **Decupagem de Produção** | Plano de câmera (close/médio/aberto), movimento, observações de set |
