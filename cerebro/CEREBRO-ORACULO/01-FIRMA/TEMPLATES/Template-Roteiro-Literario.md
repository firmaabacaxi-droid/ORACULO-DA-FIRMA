# Template — Roteiro Literário
*Formato narrativo · para documentários, institucionais e vídeos de história longa*

---

## Quando usar este formato

- Vídeos institucionais (2–3 min)
- Documentários curtos (5–10 min)
- Vídeos que dependem de narrativa, personagens ou locução em off
- Quando o cliente precisa revisar e aprovar o texto antes da filmagem

→ Para vídeos comerciais, de redes sociais ou com múltiplas peças: use [[Template-Roteiro-Tecnico]]

---

## Notas de formatação (Formato Master Scene)

```
Cabeçalho de cena → MAIÚSCULO: INT./EXT. LOCAÇÃO — DIA/NOITE
Ação → tempo presente, sem subjetividade ("a câmera mostra", não "sentimos")
Nome do personagem → MAIÚSCULO, centralizado
Diálogo → abaixo do nome, recuado
Parentética → (entre parênteses, mínimo, só quando essencial)
V.O. → voz em off (personagem fala mas não está na cena visualmente)
O.S. → off screen (personagem está fora do quadro, no mesmo espaço)
Regra de tempo → ~1 página = ~1 minuto de tela
```

## Estrutura narrativa por tipo

**Vídeo institucional (2–3 min)**
```
Cena 1 — Abertura (0:00–0:20) — gancho visual, sem palavras
Cena 2 — Contexto (0:20–0:45) — por que esta empresa existe
Cena 3 — Solução (0:45–1:30) — o que fazem e como
Cena 4 — Prova (1:30–2:00) — resultados, depoimentos, números
Cena 5 — CTA (2:00–2:20) — o que o espectador deve fazer
Cena 6 — Encerramento (2:20–2:40) — logo, assinatura
```

**Documentário curto (5–10 min)**
```
Abertura — cena de impacto que representa o tema
Apresentação — quem são os personagens, qual o contexto
Conflito — qual é o problema central
Desenvolvimento — como as pessoas lidam com ele
Virada — momento de transformação ou revelação
Encerramento — onde estão agora, reflexão
```

**Vídeo para redes (30–60s)**
```
0:00–0:03 — Gancho visual (para o scroll)
0:03–0:20 — Contexto mínimo
0:20–0:45 — Conteúdo principal
0:45–0:60 — CTA ou encerramento
```
