# Template — Plano de Rodagem
*Agenda logística das diárias · DOOD · one-liner · para filmagens com 2+ dias*

---

## Quando usar este template

- Filmagens com **2 ou mais diárias**
- Projetos com **múltiplas locações** que precisam ser agrupadas logisticamente
- Quando o **elenco ou equipe** tem disponibilidade limitada (DOOD = custo de cachê)
- Para gerar o **resumo orçamentário** da produção antes do set

→ Para um único dia de filmagem, basta a Ordem do Dia ([[Template-Ordem-do-Dia]])
