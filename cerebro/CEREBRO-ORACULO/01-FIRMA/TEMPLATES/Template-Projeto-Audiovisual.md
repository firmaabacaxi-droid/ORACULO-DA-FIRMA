# Template — Projeto Audiovisual
*Documento de desenvolvimento · logline → sinopse → argumento → escaleta*

---

## Quando usar este template

- Projetos submetidos a **editais** (ANCINE, FSA, FAC-DF, fundações culturais)
- **Documentários** e longas-metragens em desenvolvimento
- Projetos que precisam de **captação de recursos** ou coproduções
- Qualquer projeto onde o cliente precisa de um documento formal de apresentação antes do roteiro

→ Para vídeos institucionais e comerciais com roteiro direto: use [[Template-Roteiro-Literario]] ou [[Template-Roteiro-Tecnico]]
