# Template — Log de Filmagem
*Controle de takes durante o set · mais preciso que o Diário para uso na edição*

---

## Como usar

- Preencha durante a filmagem (um responsável por cena)
- Marque cada take como OK (aprovado) ou NG (reprovado)
- Marque "Válido?" = Sim apenas nos takes que devem ir para edição
- Ao final do dia, envie o log para o editor junto com o material
- Salve em: `output/roteiros/NomeCliente_log_YYYYMMDD.md`

## Legenda

| Símbolo | Significado |
|---------|-------------|
| OK | Take tecnicamente aprovado |
| NG | Take reprovado (problema técnico ou de performance) |
| Válido = Sim | Usar na edição |
| Válido = Não | Pode descartar (mas manter até entrega final) |
