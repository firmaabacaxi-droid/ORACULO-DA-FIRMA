---

## Notas nesta pasta

### Documentos de projeto (início do processo)
- [[Template-Projeto-Audiovisual]] — Logline · sinopse · argumento · escaleta · personagens · orçamento preliminar — para editais e documentários

### Documentos de pré-produção
- [[Template-Roteiro-Literario]] — Formato Master Scene: Visual / Áudio / Locução / Texto na tela — lista de personagens, locações, histórico de revisões
- [[Template-Roteiro-Tecnico]] — Formato tabular por plano: Tipo / Lente / Câmera / Ação / Áudio — com frame de storyboard e resumo geral
- [[Template-Analise-Tecnica]] — Breakdown por departamento (Fotografia, Arte, Figurino, Maquiagem, Som, Produção, VFX) — cena a cena
- [[Template-Plano-Rodagem]] — Plano por diária · DOOD (dias de elenco) · one-liner por cena · resumo orçamentário — para 2+ diárias
- [[Template-Autorizacoes-Contratos]] — Cessão de imagem (maior/menor) · autorização de imóvel · checklist de licenças (ANAC, ECAD, patrimônio)

### Documentos de set (produção)
- [[Template-Ordem-do-Dia]] — Call sheet completo: locação, meteorologia, cronograma, cenas, call times por função, refeições, transporte, emergências
- [[Template-Ficha-Continuidade]] — Raccord visual por take: figurino, maquiagem, props entre planos — diferente do Log
- [[Template-Boletim-Camera-Som]] — Metadados técnicos: lente, t-stop, ND, cartões, áudio por take, DIT checklist — para produções maiores
- [[Template-Log-Filmagem]] — Tabela de takes: Cena / Plano / OK-NG / Válido / Horário / Duração

### Documentos de processo e aprendizado
- [[Template-Aprendizado]] — Para registrar lições aprendidas em projetos
- [[Template-Referencia-Tecnica]] — Para documentar técnicas cinematográficas
- [[Template-Processo]] — Para documentar um novo processo ou fluxo
- [[Template-Briefing-Cliente]] — Perguntas padrão de briefing por tipo de projeto
- [[Template-Email-Entrega]] — E-mail formal de entrega de projeto
