# Fluxo completo de um projeto audiovisual
*13 etapas · 2 decisões críticas · fluxo sequencial*

---

## Visão geral

```
PROSPECÇÃO → CRM → CLIENTE → PRIMEIRA ABORDAGEM → ORÇAMENTO
→ PRÉ-PRODUÇÃO → PRODUÇÃO → EDIÇÃO → ACOMPANHAMENTO
→ EDIÇÃO FINAL → ENTREGA → NOTA FISCAL → PAGAMENTO
```

---

---

## Decisões críticas

**Etapa 4 — Primeira Abordagem:**
- ✅ Cliente aprova briefing → avança para Orçamento
- ❌ Não aprova → reformula e apresenta novamente

**Etapa 7 — Produção:**
- Máximo de **2 alterações** sem custo adicional
- A partir da 3ª → aditivo no orçamento (avisar cliente antes)

---

## Regras de negócio

- Projeto só entra em "Em Produção" se a Proposta estiver "Aprovada" e tarefas de pré-produção "Concluídas"
- Equipamento só é alocado se estiver "Disponível" no banco EQP
- Alertas automáticos 3 dias antes de qualquer prazo
- Projeto "Fechado" financeiramente só quando todas receitas recebidas e despesas pagas
