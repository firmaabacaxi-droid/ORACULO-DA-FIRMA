# 🚀 Como Ativar os Plugins Obsidian

Os 4 plugins foram instalados em `.obsidian/plugins/`. Agora você precisa ativá-los no Obsidian.

## ✅ Plugins Instalados

1. **HangarX** — Knowledge graph consultável por IA
2. **3D Graph** — Visualize seu cérebro em 3D
3. **Extended Graph** — Gráfico nativo estendido
4. **Advanced Canvas** — Canvas com integração a graph

---

## 🎯 Passo a Passo para Ativar

### 1. Abra o Obsidian

- Abra seu vault: `C:\Users\User\Documents\ORACULO - FIRMA ABACAXI\cerebro\CEREBRO-ORACULO`

### 2. Vá para Configurações

- **Settings** (engrenagem no canto inferior esquerdo)
- **Community Plugins** (na barra lateral esquerda)

### 3. Ative cada plugin

Você verá uma lista com os 4 plugins novos. Para cada um:

- ✅ Clique no **toggle switch** para ativar
- Ordem recomendada:
  1. **HangarX** — ativa primeiro
  2. **Extended Graph** — depois
  3. **3D Graph** — depois
  4. **Advanced Canvas** — por último

### 4. Configure (opcional)

Cada plugin terá opções em **Settings → [Nome do Plugin]**

---

## 🧠 Como Usar

### **HangarX**
- **Command Palette** (Ctrl+P) → "HangarX"
- Seu cérebro fica consultável por IA
- Perfeito para Claude acessar contexto

### **3D Graph**
- **View** menu → "3D Graph" (ou Ctrl+Shift+G)
- Visualize seu conhecimento em 3D
- Clique nos nós para explorar

### **Extended Graph**
- **View** → "Graph View"
- Acesse as novas opções de filtro
- Configure cores por tag/tipo

### **Advanced Canvas**
- Crie um novo arquivo `.canvas`
- Integra com o graph view
- Perfeito para flowcharts

---

## 🔍 Verificação

Se os plugins aparecerem em Settings → Community Plugins, está tudo certo!

Se não aparecerem:
1. Feche e abra o Obsidian novamente
2. Verifique se os arquivos `manifest.json` existem em cada pasta de plugin
3. Procure por erros no console (Ctrl+Shift+I)

---

## 🚨 Troubleshooting

**Problema:** Plugin não aparece em Community Plugins
**Solução:** 
- Verifique se a pasta tem `manifest.json`
- Rode `npm run build` manualmente na pasta do plugin

**Problema:** Obsidian travando após ativar plugin
**Solução:**
- Desative todos, ative um por vez
- Comece por HangarX
