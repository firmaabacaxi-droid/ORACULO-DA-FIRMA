# 📸 Guia Visual — Ativar Plugins no Obsidian

## Passo 1: Abrir Obsidian

Abra seu vault:
```
C:\Users\User\Documents\ORACULO - FIRMA ABACAXI\cerebro\CEREBRO-ORACULO
```

---

## Passo 2: Ir para Community Plugins

1. Clique na **engrenagem** (⚙️) canto inferior esquerdo
2. Procure por **"Community Plugins"** na barra lateral esquerda
3. Clique em **"Browse"** ou procure pelos plugins

---

## Passo 3: Ativar cada plugin

### 1️⃣ **HangarX** 
- Procure: "HangarX"
- Clique no **toggle switch** para ativar
- **O que faz:** Seu cérebro fica consultável por IA
- **Como usar:** Command Palette (Ctrl+P) → "HangarX"

### 2️⃣ **3D Graph**
- Procure: "3D Graph" (pode estar como "New 3D Graph")
- Clique no **toggle switch** para ativar
- **O que faz:** Visualizar conhecimento em 3D
- **Como usar:** View menu → "3D Graph" ou Ctrl+Shift+G

### 3️⃣ **Extended Graph**
- Procure: "Extended Graph"
- Clique no **toggle switch** para ativar
- **O que faz:** Estende gráfico nativo com filtros avançados
- **Como usar:** View → "Graph View" (as opções novas aparecem lá)

### 4️⃣ **Advanced Canvas**
- Procure: "Advanced Canvas"
- Clique no **toggle switch** para ativar
- **O que faz:** Canvas com integração a graph, flowcharts, etc
- **Como usar:** Crie um novo arquivo `.canvas`

## 🚀 Primeiro teste

Depois de ativar, tente:

1. **Abrir 3D Graph:**
   - Command Palette: Ctrl+P
   - Digite: "3D Graph"
   - Pressione Enter
   - Você vai ver seu cérebro em 3D!

2. **Testar HangarX:**
   - Command Palette: Ctrl+P
   - Digite: "HangarX"
   - Pressione Enter
   - Agora Claude consegue consultar seu conhecimento

3. **Ver Extended Graph:**
   - View menu (canto superior esquerdo)
   - "Graph View"
   - Veja as novas opções de filtro e cores

## 🆘 Se algo der errado

**Plugin não aparece em Community Plugins?**
- Feche e abra Obsidian novamente
- Verifique se a pasta existe: `.obsidian/plugins/[nome-plugin]/`
- Procure por `manifest.json` em cada pasta

**Plugin aparece mas não ativa?**
- Clique em "Install" primeiro (se não estiver instalado)
- Depois clique em "Enable"
- Reinicie Obsidian

**Obsidian travando?**
- Desative todos os plugins
- Ative um por um para identificar qual causa problema
- Comece com HangarX (é o mais importante)

---

## 💡 Dica

Se os plugins não aparecerem automaticamente, você pode:

1. Abrir Obsidian
2. Command Palette → "Community Plugins: Browse"
3. Procurar por cada nome
4. Clicar em "Install"
5. Depois clique "Enable" pra ativar
