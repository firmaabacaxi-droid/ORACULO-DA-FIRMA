# 3D Graph — Como Usar

## 🚀 Abrir em 3 passos

1. **Command Palette:** Pressione `Ctrl+P` (Windows) ou `Cmd+P` (Mac)
2. **Buscar:** Digite `3D Graph: Open`
3. **Ativar:** Pressione Enter

## 🎮 Controles (quando estiver aberto)

| Ação | Controle |
|---|---|
| **Girar o grafo** | Click + arraste do mouse |
| **Aproximar (zoom in)** | Scroll do mouse para cima |
| **Afastar (zoom out)** | Scroll do mouse para baixo |
| **Resetar câmera** | Clique direito |
| **Filtrar nós** | Clique em um nó para destacar relacionadas |
| **Procurar nó** | Use a barra de busca no painel |

## 🎨 O Que Você Verá

- **Bolinhas (nós)** = suas notas e conceitos
- **Linhas (edges)** = conexões entre conceitos
- **Cores** = diferentes tipos (projetos, clientes, etc.)
- **Tamanho dos nós** = importância (notas mais linkadas ficam maiores)

## 💡 Dicas

- Girar + zoomar ajuda a entender como seus conhecimentos se conectam
- Se vir um nó grande = essa nota é central no seu vault (muitas conexões)
- Nós isolados = notas sem wikilinks (ótimo para linkar!)
